<h2>Arquitetura e Boas Práticas</h2>
<p>No desenvolvimento das minhas aplicações, priorizo sempre a qualidade e a sustentabilidade do código:</p>
<ul>
  <li><strong>Padrões de Projeto & Arquitetura:</strong> Implementação de MVC, separação clara de responsabilidades (Controllers, Services, Repositories) e uso de padrões como DTOs (Java Records).</li>
  <li><strong>Qualidade de Código & Evolução de Dados:</strong> Aplicação rigorosa de Clean Code e princípios SOLID. Controle estrito de esquemas de banco de dados via migrações com Flyway.</li>
  <li><strong>Testes, Transações & Observabilidade:</strong> Implementação de logs estruturados, controle transacional seguro (@Transactional) para integridade dos dados e monitorização contínua.</li>
  <li><strong>Sistemas Distribuídos & Integrações Modernas:</strong> Comunicação assíncrona utilizando mensageria (RabbitMQ), consumo de serviços reativos com WebClient e integração com APIs de Inteligência Artificial.</li>
</ul>
